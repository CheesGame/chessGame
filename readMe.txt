本程序名：围棋对弈平台
实现语言：C#
开发平台：Windows 7+ Microsoft visual stdio 2012/2013
本程序作者：李琛，高健，焦海珍，张从越

首先，主文件夹chessGame打开后，
程序分为两个文件夹：{
	1. SocketTest——服务器端
	2. WindowsFormsApplication1——单机客户端
}

然后，从SocketTest打开文件夹后，
从SocketTest.csproj打开服务器项目。

然后，WindowsFormsApplication1打开文件夹后，
程序分为两个文件夹：{
	1. chessBasic——类库，棋谱算法部分
	2. clientSocket——类库，单机客户端
	3. sgf——类库，棋谱部分
	4. WindowsFormsApplication1——界面部分，为主体部分，用于连接1，2，3三个类库部分
}

最后，打开WindowsFormsApplication1文件夹，
从WindowsFormsApplication1.csproj打开主体项目。
